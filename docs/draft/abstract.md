本项目提出面向软件仓库的“Agent 驾照”机制，用于评估 Code Agent 与 LLM 工作流接入研发体系时的仓库理解能力、规范遵循能力与回归风险控制能力。与传统模型或框架评测不同，真实研发环境中的智能体通常由模型、框架、技能、规则和 Prompt 等多种要素组合而成，其最终能力高度依赖具体配置，且在不同仓库中的表现差异显著。现有 Benchmark 主要面向通用模型能力或框架效果，难以衡量智能体在特定代码仓库中的可信协作水平。针对这一问题，项目将基于 Commit 历史、Issue/PR、测试用例和工程规范，自动生成可回放、可评测、可分级的仓库级任务环境，对智能体的实际开发能力进行系统评估，并支持基于评估结果的分级授权，从而提升智能体参与软件研发的可验证性与可治理性，推动智能体从“会写代码”走向“持证上岗”。
